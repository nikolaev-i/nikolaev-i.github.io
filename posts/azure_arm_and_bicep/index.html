The Azure gym

2023-10-12
#Azure  #DevOps 

Welcome to the Azure Gym⌗

I have to give it to Microsoft; their naming game is pretty good sometimes - ARM, Bicep. It feels like I’m going to the gym rather than writing infrastructure.

So far, I’ve only used ARM templates for some tutorials, and oh boy, do I hate it - templates with JSON. People say you get used to it, but I’d rather not. Sure, everything works out of the box, and you don’t need to wait for Terraform/OpenTofu to implement new provider features.

Now, Bicep is something “new” I would’ve ignored it if it wasn’t for the changes to AZ-104 that include Bicep in the exam. To be honest, it looks a lot like TF, so it might not be that bad.

I’m using MS Learn and ACG to study, and I’ll be writing notes and code on this journey to the Azure gym.
